  # 我的大模型学习路线

  这是一个交互式的大模型学习路线图网页，帮助你系统性地规划和学习大模型（LLM）相关知识。

  ---

  ## 功能特点

  - 📊 **可视化学习路线**：以树状流程图的形式清晰展示学习路径  
  - 💡 **悬停提示**：鼠标悬停在各个节点上可查看对应的学习建议与方法  
  - ✅ **进度追踪**：每个学习节点支持勾选，便于记录学习完成情况  
  - 💾 **本地存储**：学习进度自动保存至浏览器本地存储，刷新或重开页面不会丢失  
  - 🎨 **美观界面**：现代化渐变背景与流畅动画效果  
  - 📱 **响应式设计**：支持桌面端与移动端访问  

  ---

  ## Quick Start（快速启动）

  ### 方法一：一键启动（推荐）

  项目已提供启动脚本，可直接在本地启动一个 HTTP 服务：

  ```bash
  bash start_server.sh
  ```

  然后在浏览器中访问：

  ```
  http://localhost:8000
  ```

  > 该方式最接近真实部署环境，推荐作为日常使用方式。

  ---

  ### 方法二：直接打开（最简单）

  1. **双击 `index.html` 文件**
     - 文件会在默认浏览器中自动打开  
     - 无需安装任何依赖或启动服务器  

  2. **或右键点击 `index.html`**
     - 选择“打开方式” → 选择你喜欢的浏览器  

  ---

  ### 方法三：使用本地服务器（可选）

  #### 使用 Python（如果已安装）

  ```bash
  # Python 3
  python -m http.server 8000

  # Python 2
  python -m SimpleHTTPServer 8000
  ```

  浏览器访问：

  ```
  http://localhost:8000
  ```

  #### 使用 Node.js（如果已安装）

  ```bash
  npm install -g http-server
  http-server -p 8000
  ```

  浏览器访问：

  ```
  http://localhost:8000
  ```

  #### 使用 VS Code

  - 安装 **Live Server** 扩展  
  - 右键点击 `index.html` → 选择 **Open with Live Server**

  ---

  ## 使用方法

  1. 在浏览器中打开页面  
  2. 鼠标悬停在各个学习节点上查看详细的学习建议  
  3. 完成某一学习阶段后，勾选对应节点进行标记  
  4. 学习进度会自动保存，下次打开页面时自动恢复

  ## 学习路线内容

  本项目的整体学习路线设计，参考并整理自：

  > **清华姜学长（B 站 UP 主）**  
  > 《5分钟讲清楚 大模型学习路线》  
  > *#大模型 #AIGC #深度学习 #智能体 #算法*

  在此基础上，对学习阶段进行了结构化与工程化整理，形成如下学习路径：

  1. **初级目标**：会调用主流大模型 API  
  2. **中级目标**：理解模型训练与优化，使用 LoRA 进行指令微调  
  3. **高级目标**：模型优化与部署，开发对话系统和 Agent  
  4. **理论打底**：深度学习基础与语言模型演进  
  5. **模型认知**：BERT / GPT / LLaMA 的结构与设计理念  
  6. **训练与对齐**：数据准备、训练流程、对齐技术  
  7. **工程化能力**：模型压缩、分布式训练、推理部署  
  8. **前沿应用**：RAG 系统、Agent 系统、多模态能力
