# 我的大模型学习路线

这是一个交互式的大模型学习路线图网页，帮助你系统性地规划和学习大模型相关知识。

## 功能特点

- 📊 **可视化学习路线**：清晰的树状流程图展示学习路径
- 💡 **悬停提示**：鼠标悬停在每个节点上可查看详细的学习方法
- ✅ **进度追踪**：每个学习点都有复选框，完成学习后可标记
- 💾 **本地存储**：学习进度自动保存到浏览器本地存储
- 🎨 **美观界面**：现代化的渐变背景和流畅的动画效果
- 📱 **响应式设计**：支持桌面和移动设备访问

## Quick Start（快速启动）

### 方法一：直接打开（最简单）

1. **双击 `index.html` 文件**
   - 文件会在你的默认浏览器中自动打开
   - 无需安装任何依赖或启动服务器

2. **或者右键点击 `index.html`**
   - 选择"打开方式" → 选择你喜欢的浏览器

### 方法二：使用本地服务器（推荐）

如果你想通过 `http://localhost` 访问，可以使用以下任一方法：

**使用Python（如果已安装）：**
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```
然后在浏览器访问：`http://localhost:8000`

**使用Node.js（如果已安装）：**
```bash
# 安装 http-server（全局安装，只需一次）
npm install -g http-server

# 在项目目录运行
http-server -p 8000
```
然后在浏览器访问：`http://localhost:8000`

**使用VS Code：**
- 安装 "Live Server" 扩展
- 右键点击 `index.html` → 选择 "Open with Live Server"


## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 鼠标悬停在各个学习节点上查看详细的学习方法
3. 完成学习后，点击节点右侧的复选框进行标记
4. 学习进度会自动保存，下次打开时会自动恢复

## 学习路线内容

1. **初级目标** - 会调用API
2. **中级目标** - 理解模型的训练和优化，会用LoRA去做一些指令微调
3. **高级目标** - 独立模型的优化和部署能力，开发对话系统和Agent
4. **理论打底** - 回顾深度学习的知识，语言模型的整个演进
5. **模型认知** - 搞懂BERT、GPT、LLaMA的结构差异和设计理念
6. **训练和对齐** - 数据准备、训练过程、微调阶段、对齐技术
7. **工程化** - 模型压缩、分布式训练、推理部署
8. **聚焦前沿的应用** - RAG系统、Agent系统、多模态能力
